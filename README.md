Multi-Agent Coordination in Networked Games

This project explores Multi-Agent Reinforcement Learning (MARL) using Q-Learning to solve a coordination game across a specific adjacency graph. We analyze how agents transition from "blind" independent learning to state-aware strategic coordination.

📂 Project Structure

`report.ipynb`: The main technical report containing the final analysis, math, and combined visualizations.

`scenario1.py`: Implementation of independent Q-Learning where agents have no vision of their neighbors.

`scenario2.py`: Implementation of state-based Q-Learning where agents observe neighbors' previous actions.

`scenario1_*.png`: Resulting plots for Scenario 1 (Action Values and Mean Rewards).

`scenario2_*.png`: Resulting plots for Scenario 2 (Action Values and Mean Rewards).
